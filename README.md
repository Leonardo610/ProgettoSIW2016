Questo è il repository finale del progetto per il corso di Sistemi Informativi su Web del prof. Paolo Merialdo dell'Università degli Studi Roma Tre svolto in collaborazione con i colleghi Giovanni Antonio Me e Marco Bellocchi.

Consiste in un'applicazione web su Tomee che offre servizi di login, prenotazione, consultazione, ricerca, filtri e amministrazione di prestazioni sanitarie offerte da una fittizia ClinicaACME con base di dati persistente.
